/*Propiedades y tamaño del tablero de cristal*/
.tablero-cristal {
  position: relative;
  min-width: 500px;
  max-width: 100vh;
  height: 350px;
  margin: auto;
  background: inherit;
  border-radius: 5px;
  overflow: hidden; /* ocultamos el efecto de desenfoque si se sale de la caja */
  box-shadow: inset 20px 20px 200px -70px #B3B6B7; /*sombra interior, asi logramos un efecto mas cercano a un cristal*/
}

/* Efecto de cristal opaco */
.tablero-cristal::after {
  content: '';
  width: 100%;
  height: 100%;
  background: inherit; 
  position: absolute;
  top: 0;
  filter: blur(10px);
}

/* Sobreponemos el contenido por delante del desenfoque */
.contenido-cristal {
  padding: 30px;
  position: relative;
  z-index: 1;
}

Para conseguir una sombra interior debemos añadir inset a box-shadow asi conseguimos un efecto mas cercano al cristal opaco. Para poder conseguir el tablón con esta textura debemos tener un div y añadira la clase la propiedad after, ej: .tablero-cristal::after y aplicar un desenfoque(filter: blur(10px)). Es importante heredar el background para que el efecto funcione.

Dependiendo del la posición que le demos a las pestañas (ej: relative) se pueden superponer al tablero usando un z-index cuando la pestaña esté activa, las demás estarán por detrás y se pueden posicionar moviendolas por ejemplo con bottom, esto combinado con un border-bottom (ej: elegir el color de fondo del tablero del mismo color) nos permite un efecto que al pinchar en la pestaña se combinen pestaña y tablero, no se ve el borde inferior pero si los laterales.
Para el tablero: determinar la distancia entre los margenes del elemento y de su bloque contenedor usamos las propiedades left, rigth, bottom y top, así cada pestaña mantiene su posicion y oculta a las demás, si queremos un efecto en el que se vean una encima de otra y se vea un poco el borde podemos quitar estas propiedades o darles valores diferentes

Explicación

Nota: el código completo está en la otra pestaña

Para empezar creamos un contenedor con texto dentro, en nuestro caso un h3. El h3 contendrá la imagen, pero se la añadiremos a la clase para hacer baners diferentes, le damos un tamaño del 100% para que los extremos bajen igual hacia el centro por la perspectiva del contenedor padre con perspective: 800px (como si observaramos desde el frente), si le damos un tamaño de la mitad o inferior conseguimos un efecto como si observaramos un poco de lado.

                    
                    /* Contenedor principal */
                    .cuadro1 { 
                      perspective: 800px; 
                      margin-bottom: 10%; /*Añadimos un margen inferior por que las rotaciones 
                      y sombra ocupan posiciones fuera del contenedor, si hay algo debajo no interfieren*/
                    }

Dentro del contenedor tendremos un texto al que le daremos estilos (y añadimos una imagén) y usamos la propiedad transform de CSS3 que nos da la posibilidad de transformar elementos en 2D o en 3D. Nos permite rotar, escalar, mover, torcer, etc, elementos.

                    
                    /* Contenedor principal */
                    .cuadro1 h3 {
                      transform-style: preserve-3d;
                      transition: all 0.5s; /*A partir de 0.6 la sombra rota de más */
                    }

En .cuadro h3::after{} procedemos a añadir la cita o si creamos una clase para cada cuadro se la añadimos a este y obtendremos varias frases.

El contenido del h3 lo tenemos en un párrafo para que no interfiera cuando movamos las capas y poder hacerlo independientemente. Capa de sombra (::before), capa con autor (p), capa con imagen (h3) y capa con la frase del autor (::after). En .cuadro h3 p utilizamos transform: rotateX(-90deg) para rotar de manera que el texto esté mirando hacia abajo, modificamos el origen de la transformacion del parrafo que está justo debajo de la imagen, ya tumbado por la rotación. Al darle origen top empieza desde arriba y a la vez conseguimos que se suba y se oculte detrás de la imagen antes de iniciar el efecto.

                    
                    .cuadro1 h3 p { 
                      position: relative;
                      transform-origin: top;
                      transform: rotateX(-90deg); /* lo rotamos de manera que el texto esté mirando hacia abajo*/
                    }

En .cuadro1:hover h3{} para conseguir que la parte inferior no se mueva y se tumbe desde arriba originamos la rotacion desde abajo

                    
                      /* Rotación del h3 (imagen y texto) */
                      .cuadro1:hover h3 {
                        transform: rotateX(75deg);
                        transform-origin: bottom; 
                      }

Para la sombra hacemos uso de ::before en el h3, y le damos una posición absoluta ya que por ejemplo la posición relativa nos hace la sombra en el punto 0,0 del .cuadro por lo que necesitamos que sea absoluta o fixed. Con transform: rotateX(95deg) translateZ(-50px) scale(0.75) creamos una sombra, la tumbamos al rotarla en el eje X y y la profundidad (z) en negativo para bajarla.

                    
                    .cuadro1 h3::before {
                      content: "";
                      position: absolute; 
                      box-shadow: 0 0 100px 50px rgba(0, 0, 0, 0.45);
                      transition: all 0.5s;
                      transform: rotateX(95deg) translateZ(-50px) scale(0.75);  
                      transform-origin: bottom; 
                    }

Por último, para que la animación no se dañe, el ::before del h3 contiene la sombra girada por lo que al posicionar el ratón encima se moverá, entonces debemos darle una posición y un estilo, todo ello lo haremos en: .cuadro:hover h3::before{} 

                    
                    .cuadro1:hover h3::before {
                      opacity: 0.5;
                      transform: translateZ(-50px) scale(0.9);            
                    }
